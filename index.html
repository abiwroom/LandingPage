01 Kesan Semester Ganjil

Semester ganjil ini memberikan banyak pengalaman baru bagi saya, terutama dalam proses pembelajaran di kelas XI RPL 1. Setiap mata pelajaran memiliki tantangan masing-masing yang melatih kedisiplinan dan tanggung jawab saya terutama dalam mengerjakan tugas-tugas yang diberikan.

Khususnya pada mata pelajaran yang diajar oleh Pak Unggah. Cara mengajar yang tegas namun jelas membuat saya terdorong untuk memahami materi dengan lebih baik. Berbeda dengan guru lain, di mapel ini setiap pertemuan harus ada catetan materi yang ditulis tangan, yang awalnya saya berpikir kenapa harus mencatat ulang padahal materi sudah ada di buku digital yang diberikan. Namun seiring waktu saya menyadari bahwa dengan saya mencatat materi yang sudah diberikan, saya lebih mudah memahami dan mengingatnya dan ketika saya lupa beberapa code saya bisa melihat catatan saya kembali.

Walaupun terkadang terasa sulit, saya merasa bangga bisa melewati semester ini dengan baik namun masih banyak hal yang harus saya perbaiki dan tingkatkan seperti lebih konsisten dalam belajar terutama javascript. Karena menurut saya javascript adalah materi yang paling sulit untuk saya pahami terutama pada saat projek game.

Selain di mapel KKPW, di mapel KKPG saya merasa susah memahami materi karena saya baru pertama kali mendengar materi yang diberikan namun saya berusaha untuk memahami materi tersebut dengan mempelajari baik dari pdf yang diberikan atau belajar dari youtube. Saya berharap di semester berikutnya saya bisa lebih baik lagi dalam memahami materi-materi yang diberikan.

Secara keseluruhan, semester ganjil ini mempunyai pengalaman yang berharga bagi saya. Saya belajar banyak hal baru, tidak hanya tentang teknologi dan pemrograman, tetapi juga tentang bagaimana mengatur waktu dan bekerja dalam tim.

Selain dalam proses pembelajaran, pada saat hut skensa saya berpartisipasi untuk memeriahkan acara menampilkan tari sekar jagat sebagai tari pembukaan. Saya juga mengurus lomba seperti memilih peserta lomba dan membantu jegeg bagus.

02 Kegiatan Selama Liburan

Saat liburan semester, saya memanfaatkan waktu untuk beristirahat karena saya merasa saya sudah melewati banyak hal yang dulu saya merasa saya tidak mungkin sampai ke titik ini. Liburan menjadi momen untuk melepaskan rasa lelah setelah menjalani kegiatan belajar yang padat. Saya menghabiskan waktu bersama teman-teman saya diantaranya keluar untuk makan, photobooth, dan tentunya hal wajib yaitu jalan jalan ke living world. Tetapi saya lebih sering berada dirumah, karena saya merasa setiap keluar banyak uang yang saya keluarkan tanpa memikirkan kedepannya.

Selain itu, saya juga mempunyai usaha bouqet bunga untuk menambah uang jajan saya. Walaupun tidak terlalu banyak yang terjual tapi saya meresa senang karena saya bisa menjalankan hobi saya sekaligus mendapatkan uang jajan tambahan.

03 Harapan di Semester Genap

Di semester genap nanti, saya berharap dapat lebih fokus dan konsisten dalam belajar. Saya ingin meningkatkan pemahaman di semua mata pelajaran, khususnya yang berkaitan dengan pemrograman. Karena di semester lalu saya merasa kurang dalam memahami materi terutama di javascript dan database selain mysql.

Saya juga berharap dapat lebih aktif di kelas, tidak ragu untuk bertanya dan menjawab, dan mampu mengatur waktu dengan lebih baik antara belajar dan kegiatan lainnya.

04 Cerita Semester 2

Sebagai siswi kelas 11 yang mengambil jurusan rekayasa perangkat lunak saya menyadari bahwa kita harus selalu mengikuti perkembangan zaman yang dimana ketika kita tidak mau mempelajari suatu aplikasi dengan baik dan benar kita akan ketinggalan banyak hal, karena saya pernah merasakan hal tersebut ketika teman saya sudah bisa menguasai berbagai aplikasi yang bersangkutan dengan website sedangkan saya masih stuck di 1 aplikasi saja. Disana saya muncul pemikiran bahwa, saya harus mencoba untuk mempelajari banyak aplikasi yang berhubungan dengan website. dalam hal coding pun saya menyadari bahwa menguasai Visual Studio Code bukan sekadar tempat mengetik sintaks, tetapi bagaimana memanfaatkan berbagai extension untuk mempercepat alur kerja. Melihat mereka begitu lancar menggunakan framework seperti Bootstrap untuk mempercantik tampilan website, sementara saya masih berkutat dengan CSS murni, membuat saya tersadar. Dunia rekayasa perangkat lunak bergerak sangat cepat; hari ini sebuah teknologi menjadi tren, besok bisa jadi sudah tergantikan. Ketertinggalan ini menjadi tamparan keras sekaligus motivasi bagi saya bahwa menjadi siswi RPL berarti harus siap menjadi pembelajar seumur hidup. Saya tidak boleh hanya puas dengan satu aplikasi atau satu metode saja, karena esensi dari jurusan ini adalah inovasi dan adaptasi terhadap alat-alat baru yang terus bermunculan.

Pada liburan 2 minggu yang bertepatan dengan libur hari raya nyepi dan libur hari raya idul fitri, sehari sebelum hari raya nyepi ada hari pengrupukan pada hari pengrupukan tentu identik dengan ogoh-ogoh yang dimana ogoh ini berfungsi sebagai simbolis penyucian diri dan lingkungan, yang akhirnya akan dibakar untuk memusnahkan kejahatan agar umat Hindu dapat menjalani Catur Brata Penyepian dalam keadaan suci. Pada hari pengrupukan saya ikut meramaikan dengan mengikuti pawai ogoh ogoh yang dimana saya menari cendrawasih di pawai ogoh-ogoh tersebut. Beberapa minggu sebelumnya saya disibukkan dengan latihan untuk pawai ini, tentu saya jadinya pulang malam terus-terusan. Dan pada saat pengrupukan saya mulai make-up sekitar jam 3 sore padahal saya sudah datang sekitar jam 10 pagi. Pawai dimulai di jam 7 malam, dan Banjar saya mendapatkan urutan ke 3 yang dimana sudah tampil sekitar jam 8an. Semua kelelahan saya dibayar dengan pawai ogoh-ogoh Banjar saya mendapatkan juara 2.

Sehari sebelum hari pengrupukan saya membeli makanan untuk stok saya di hari raya nyepi, tidak terlalu banyak hanya beberapa saja karena saya berpikir bahwa saya tidak akan terlalu banyak memakan jajan tersebut di hari raya nyepi, pada saat nyepi yang saya duga pun terjadi dimana saya membeli sekitar 8 jajan dan yang saya makan hanya 3 jajan saja. Di hari raya nyepi ini berbeda dengan nyepi sebelumnya karena pada saat ini kakak saya harus menginap di hotel karena dia tidak mendapatkan libur untuk hari raya nyepi. Di hari raya nyepi ini juga saya membersihkan kamar dan lemari saya, membuang barang barang yang sudah tidak saya butuhkan.

Setelah hari raya nyepi, selanjutnya ada hari raya idul fitri di hari raya idul fitri ini saya hanya diam dirumah saja karena keluarga saya sibuk bekerja tetapi saya tidak terlalu memikirkan hal tersebut karena mereka juga bekerja untuk saya agar bisa menyekolahkan saya. Saya menghabiskan waktu liburan saya dengan membuat orderan bouquet kawat bulu saya lalu saya juga bermain dengan kucing saya yang saya beri nama mila. Mila sering menemani saya dalam membuat bouquet kawat bulu, walaupun kadang-kadang saya kesal dengan mila karena dia sering memainkan kawat bulu saya. Meskipun liburan ini diwarnai dengan kesibukan keluarga dan kesibukan membuat bouquet, saya merasa jauh lebih siap untuk kembali ke sekolah. Pengalaman liburan ini mengajarkan saya tentang keseimbangan antara menjaga tradisi budaya, tanggung jawab pekerjaan, dan ambisi saya untuk terus maju menguasai teknologi di jurusan RPL agar tidak lagi merasa tertinggal.
